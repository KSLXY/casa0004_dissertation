# casa0004_dissertation

This repository contains the reproducible pipeline and Streamlit dashboard for Casa0004 dissertation experiments.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run reproducible_pipeline/streamlit_app.py
```

## Included Data Artifacts

The repository includes lightweight run artifacts for `run_20260415_105036` (metrics, ablation, importance, reports, figures, clustering artifacts, neighbor map) but excludes very large prediction/split files.
